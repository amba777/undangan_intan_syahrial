## 🚀 Cara Deploy ke Streamlit Cloud (GRATIS)

### Langkah 1 — Buat akun GitHub
Buka https://github.com dan buat akun jika belum punya.

### Langkah 2 — Upload ke GitHub
1. Buat repository baru (klik **New repository**)
2. Nama repo: `undangan-intan-syahrial`
3. Upload semua file dari folder ini (app.py, photos.json, requirements.txt)

### Langkah 3 — Deploy di Streamlit Cloud
1. Buka https://share.streamlit.io
2. Login dengan akun GitHub
3. Klik **New app**
4. Pilih repository: `undangan-intan-syahrial`
5. Main file path: `app.py`
6. Klik **Deploy!**

### Langkah 4 — Bagikan Link
Setelah deploy selesai (~2-3 menit), Anda mendapat link seperti:
```
https://undangan-intan-syahrial.streamlit.app
```
Link ini bisa dibagikan ke semua tamu via WhatsApp! 🎉

## 💻 Cara Jalankan di Komputer Lokal

```bash
# Install dependencies
pip install -r requirements.txt

# Jalankan aplikasi
streamlit run app.py
```

Buka browser di: http://localhost:8501
